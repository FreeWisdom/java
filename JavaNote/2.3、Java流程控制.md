# 2、顺序结构

- 语句与语句之间，框与框之间是按从上到下的顺序进行的，它是由若干个依次执行的处理步骤 组成的。
- JAVA的基本结构就是顺序结构，除非特别指明，否则就按照顺序一句一句执行。
- 它是任何一个算法都离不开的一种基本算法结构。
