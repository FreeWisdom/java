# 1、Java三大版本

* JavaSE：标准版（桌面程序，控制台......）
* JavaME：嵌入式开发（手机，小家电......）
* JavaEE：E企业级开发（web端，服务器开发......）

# 2、JDK JRE JVM

* JDK：Java开发者工具
  * JRE：Java运行时环境
    * JVM：Java虚拟机

# 3、Java开发环境搭建

## 3.1、卸载JDK

1. 删除Java安装目录；
2. 删除JAVA_HOME；
3. 删除path下关于Java的目录；
4. java-version查看版本是否删除；
